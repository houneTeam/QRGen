# QRGen

This Python script generates a QR code from raw hex data extracted from フレンダピック (Frenda Pick).

## QR Code Generation Parameters

- `version = 2`
- `error_correction = ERROR_CORRECT_M`
- `box_size = 10`
- `border = 4`
- `mask_pattern = 2`
